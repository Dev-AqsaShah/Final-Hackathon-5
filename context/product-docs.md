# TechNova Platform - Product Documentation

## Getting Started

### Account Setup
1. Sign up at app.technova.io
2. Verify your email address
3. Create your first workspace
4. Invite team members via Settings > Team > Invite Members
5. Choose your plan (free 14-day trial available)

### Password Reset
1. Go to app.technova.io/login
2. Click "Forgot Password"
3. Enter your registered email address
4. Check your inbox for the reset link (valid for 24 hours)
5. Click the link and create a new password (min 8 chars, 1 uppercase, 1 number)

**If you don't receive the email:**
- Check spam/junk folder
- Ensure you're using the correct email address
- Wait up to 5 minutes, then try again
- Contact support if issue persists

## Task Management

### Creating a Task
1. Open a project
2. Click "+ New Task" or press "N"
3. Enter task title, description, assignee, due date
4. Set priority: Low / Medium / High / Critical
5. Click "Create Task"

### Assigning Tasks
- Click on a task to open it
- Click "Assignee" field
- Search for team member by name or email
- Multiple assignees supported (Professional+)

### Task Status Workflow
- **To Do** → **In Progress** → **In Review** → **Done**
- Custom statuses available in Professional and Enterprise plans

## Time Tracking

### Starting the Timer
1. Open any task
2. Click the ⏱️ Timer icon
3. Timer starts automatically and logs to the task

### Manual Time Entry
1. Open a task
2. Click "Log Time"
3. Enter hours and minutes
4. Add a description (optional)
5. Click "Save"

### Viewing Time Reports
- Go to Reports > Time Tracking
- Filter by project, team member, or date range
- Export to CSV or PDF

## Integrations

### Slack Integration
1. Go to Settings > Integrations > Slack
2. Click "Connect to Slack"
3. Authorize TechNova in your Slack workspace
4. Choose which channels to receive notifications
5. Configure notification types (task assigned, due soon, completed)

### GitHub Integration
1. Go to Settings > Integrations > GitHub
2. Connect your GitHub account
3. Link repositories to projects
4. Commits with task IDs (#TASK-123) auto-link to tasks

### Google Workspace
1. Settings > Integrations > Google
2. Connect Google account
3. Sync Google Calendar with task deadlines
4. Import files from Google Drive

## API Access (Professional & Enterprise)

### Getting Your API Key
1. Go to Settings > Developer > API Keys
2. Click "Generate New Key"
3. Name your key and set permissions
4. Copy the key immediately (shown only once)

### API Rate Limits
- Professional: 1,000 requests/hour
- Enterprise: 10,000 requests/hour

### Webhook Setup
1. Settings > Developer > Webhooks
2. Click "Add Webhook"
3. Enter your endpoint URL
4. Select events to subscribe to
5. Test with "Send Test Event"

## Billing & Subscriptions

### Upgrading Your Plan
1. Go to Settings > Billing
2. Click "Upgrade Plan"
3. Select your new plan
4. Enter payment details
5. Changes take effect immediately

**Note:** For custom Enterprise pricing, contact sales@technova.io

### Cancellation Policy
- Cancel anytime from Settings > Billing > Cancel Subscription
- Access continues until end of billing period
- Data retained for 90 days after cancellation
- **Refund requests must be handled by billing team** - AI cannot process refunds

### Invoice & Receipts
- Settings > Billing > Invoice History
- Download PDF invoices
- Set up automatic email receipts

## Troubleshooting

### App Not Loading
1. Clear browser cache (Ctrl+Shift+Delete)
2. Try incognito/private window
3. Check status.technova.io for outages
4. Try a different browser
5. Disable browser extensions

### Notifications Not Working
1. Check browser notification permissions
2. Settings > Notifications > verify preferences
3. Check email spam folder
4. Ensure email is verified in account settings

### Data Export
1. Settings > Data > Export
2. Choose export format (CSV, JSON, PDF)
3. Select date range
4. Export sent to your email within 15 minutes

## Security & Privacy

### Two-Factor Authentication (2FA)
1. Settings > Security > Two-Factor Auth
2. Choose method: Authenticator App or SMS
3. Follow setup wizard
4. Save backup codes in a safe place

### Team Permissions
- **Owner:** Full access, billing management
- **Admin:** Manage team, projects, settings
- **Member:** Create and edit tasks, view all projects
- **Viewer:** Read-only access (Enterprise only)

### Data Security
- All data encrypted at rest (AES-256) and in transit (TLS 1.3)
- SOC 2 Type II certified
- GDPR compliant
- Regular security audits
